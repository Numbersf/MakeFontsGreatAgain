屏蔽字体
格式:
XXX.ttf 举例:
Private-UseTest.ttf 输入一个字体文件(ttf/otf/ttc)，自动重命名为 .disabled 禁用，重启生效。禁止对主字体进行此操作
屏蔽字体单区间
格式:
XXX.ttf,(U+XXXX-U+XXXX) 举例:
BraillePatterns.ttf,(U+2800-U+28FF) Unknown-symbol-supplementRegular.ttf,(U+EE01-U+EE01) 屏蔽一个字体文件单个被指定的符号范围，其他部分将继续生效
使用 Unicode 编码，编码从小到大排列。必须是此字体文件里有的范围才能处理。可以仅处理一个 Unicode，但是格式不能变。还原时只输入字体文件名即可，不需要加范围。
屏蔽字体多区间
格式:
XXX.ttf,(U+XXXX-U+XXXX;U+XXXX-U+XXXX) XXX.ttf,(U+XXXX-U+XXXX;U+XXXX-U+XXXX;U+XXXX-U+XXXX) ••• 举例:
Private-UseTest.ttf,(U+F002B-U+F005B;U+F006B-U+F007C) Private-UseTest.ttf,(U+F002B-U+F005B;U+F0183-U+F01AC;U+F0200-U+F0300) ••• 屏蔽一个字体文件多个被指定的符号范围，其他部分将继续生效
使用 Unicode 编码，编码从小到大排列。你可以通过";"分离多个区间，区间也从小到大排列，且各区间尽量避免相交。必须是此字体文件里有的范围才能处理。还原时只输入字体文件名即可，不需要加范围。
屏蔽 GMS 字体
暂停 GMS 自带的字体服务并删除本地缓存字体文件使部分 GAPP 指向系统字体。执行时会强行停止(杀后台) Gmail 和 Chrome 一次。执行后出现应用闪退请重启手机。理论上所有 Android8+ 设备都能正常工作。隔一段时间字体就会再次缓存出来，需再次执行
